# 科研团队能力图谱原型

这是一个基于 `AntV G6` 的静态原型网页，用来展示科研成员、研究方向、技术方法、仪器设备和软件工具之间的关系。

## 当前已完成

- 人物节点点击展开
- 右侧详情卡片
- 共享能力自动着色
- 搜索与类型筛选
- 可替换示例数据

## 如何录入真实信息

直接编辑同目录下的 `data.js` 文件，按示例格式为 `people` 数组增加成员：

```js
{
  id: "person-new",
  name: "成员姓名",
  role: "硕士研究生",
  group: "所属方向组",
  bio: "个人简介",
  tags: ["标签1", "标签2"],
  directions: ["研究方向1"],
  skills: ["技术1", "技术2"],
  equipment: ["仪器1", "仪器2"],
  software: ["软件1", "软件2"]
}
```

## 打开方式

直接打开 `index.html` 即可预览。

如果浏览器因本地脚本策略导致加载受限，也可以在当前目录启动一个简单静态服务器后访问。
